手机root

什么是root

        我们平时使用手机，是作为一个使用者来做的，只有部分的权限。root，翻译为根，就是从手机的底层出发，可以获得所有的权限。举个例子，租房子，就只能使用房间里给你安排好的东西，但你不能装修房子。但如果这房子是你买的，那就可以对它为所欲为了。

        手机root，就是让你获得最高的权限。你可以做到一些普通用户无法做到的事情，比如删除系统自带软件，删除系统文件，修改应用数据，手机定位，cpu超频等。

root的坏处

        root有哪些好处，大家最好自己体验一番，反正用过的都说好，但必须要先知道root的坏处。我认为只有一点：不安全。

        首先对于用户来说，用root删除某些文件导致无法开机，甚至数据丢失，是常有的事情。所以进行刷机前一定要备份，并且学会救砖。

        另外就是root权限被其他人利用。比如用root权限，获取你的私人数据，甚至锁屏密码和支付密码。再或者控制你的手机去做什么违法的事情，都是有可能的。

        第二点或许有点扯，但第一点确实是常有的事。所以root手机，也千万要小心，不然出现点什么意外就只能重刷了。     

Magisk

什么是magisk

        magisk是中国台湾的topjohnwu做的一款软件，它能获取和管理root权限，并且挂载各种扩展模块，非常的强大。

        除此之外，还有Magiskhide（现阶段只有zygisk版才有这个功能），对单个应用隐藏root权限。这一功能非常重要，屏蔽支付宝和银行app对root的root检测。

        magisk的亮点还是在于那些模块。要刷模块的话建议去酷安，xda论坛里去找。博主首推一个，自动神仙救砖。以防刷错模块导致开不了机。

Magisk原理

        magisk会对boot进行修补，那么启动的时候，boot就不会前往system分区，而是进入magisk分区。这个分区会挂载一个在system的基础上能够替换，增加和删除自定义内容的文件系统，但不会对system分区进行任何修改。所有操作在启动的时候就完成了。

        相比于以前的root软件，比如supersu，对系统进行更改，magisk更为安全，强大。目前magisk是主流的root方式。基本玩机佬的手机里都有一个。

        当卸载magisk后，系统依然完好，仍然可以进行ota升级。如果带着magisk进行ota升级的话，需要刷全量包，并且升级完成之后，要重新刷入magisk。

Magisk刷入

  • 准备工作

  1. 首先要确保手机已经解锁了bootloader。每个手机都有对应的方法，建议去官网查。以小米手机为例，下载小米解锁工具，按照上面的说明进行操作。一个小米账号一年最多解锁两个手机。

  2. 然后进入系统设置，多次点击版本号，进入开发者模式。找不到可以搜索。把开发者模式中的usb调试打开。这一步方便从系统进入bootloader，会进的可以不做。

  3. 电脑下载platform-tools,下载对应自己电脑的版本。然后解压，在文件夹中打开右键，在终端中打开。

  4. 通过数据线连接手机和电脑，在windows终端中输入adb devices，手机点允许。整个刷机过程中，手机与电脑保持连接状态。

  • 修补boot刷入（仅高通）

        这是目前比较常用并且简单的方法。首先是在刷机包中找到boot.img文件。刷机包可以在手机的系统更新中下载，也可以去官方下载对应自己版本的rom包。比如小米手机，如下图

        

        小米手机可以去小米rom中下载对应rom包。下载好后，解压，找到boot.img。如下图。

        我这里是卡刷包，线刷包里也可以找到。

  1. 将boot.img复制到手机中（留一份在电脑备用）

  2. 下载并安装magisk官方地址 ，酷安里同样可以下载。

  3. 打开magisk，安装，选择并修补一个文件，选择复制到手机中的boot.img

  4. 将得到的img文件传回电脑中。一般文件在download文件夹中，注意看magisk修补完的说明

  5. 重启手机至bootloader。可以用adb命令 adb reboot bootloader。或者关机，然后按照对应的组合按键进入。

  6. 电脑在之前打开的windows终端中，输入fastboot flash boot ***.img ***.img指修补的img文件。只要将文件拖入黑框中，就可以出现路径。后面都可以这样做。

  7. 刷写成功后重启。如果开机，就成功了。如果无法开机，就按照第六步的命令，刷入之前的boot.img文件。

联发科机型还需要刷入vemeta.img，感兴趣的可以去小米社区看看。

  • 第三方rec刷入。

  1. 首先去twrp找到适合自己的rec文件，如果没有可以尝试去xda论坛或者百度一下，有无大佬做对应的rec。

  2. 进入bootloader，输入fastboot flash recovery recovery.img recovery.img是指你下载下来的img文件。

  3. 无论第二步是否成功，你都可以输入fastboot boot recovery.img进入recovery。有些机型无法刷入rec。可以通过这种方式进入临时的rec。

  4. 将之前下载好的magisk.apk文件，更名为magisk.zip，传入手机中。

  5. 在rec模式下，点安装，选择magisk.zip文件，刷入，重启至系统。

  6. 如果成功的话，手机应用里会有一个magisk的应用。可以将magisk.zip重新更名为magisk.apk，安装到手机中。

有些大佬制作的rec中，高级选项中，有root的选项，与刷入magisk是一样的。

  • magisk模块刷入

        现在magisk的模块商店好像是没有了，只能通过本地安装了。这里推荐使用搞机助手在里面选择模块直接刷入。具体刷入什么模块，去酷安查一下就了解了。我个人不喜欢花里胡哨的东西，所以模块安的并不多。还是要大家自己折腾吧。

总结

最后，刷机前要备份！备份！备份！。刷机有风险，操作需谨慎。折腾虽好，但要注意休息，争取在一点前睡。
