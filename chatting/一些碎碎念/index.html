建立博客的缘由

　　作为一个刷机党，喜欢折腾是我的一大特点。在刷机的过程中，看过很多大佬写的教程和视频，也得到过很多网友的帮助。投之以桃，报之以李。我也应当，为那些和曾经的我一样的小白们，献上自己的绵薄之力。

　　其实不单单是刷机相关的知识。我希望我学到的一些知识，一些理念能够帮助到大家，当不了正面教程，也可以做个反例嘛。我认为，只有与他人的思想进行交流，才能提升自己。闭门造车是绝不可能得出好东西的。

　　另外，写博客还有一个原因，就是激励自己。博主现在刚毕业，准备考研中，可是总是找不到状态。三天打鱼，两天筛网，都是常有的事。也许，见个博客，写点什么，能让自己好一些，我是这么认为的。

　　知识总是一个输入到输出的过程，只有输入，脑子里不过一团浆糊，只有输出，产出的也是一团浆糊。无论哪一种方向，如果过于极端，都是学不下去的。

　　博客里的文章，与其说是教程，倒不如说是个人笔记。只提供个人思路，有问题欢迎指正。如果误人子弟了，那我真的真的非常抱歉。

博客现状及规划

　　目前博客是由 hexo 制作，用的 keep 主题。相比于以前写博客用的 Gridea，自定义化较高，但是难度也较高，不适合纯新手。不论是从，与 github 建立连接，还是下载安装，Gridea 都是极为简单和方便的。

　　还有就是目前的评论系统，用的是腾讯云的 Twikoo。用的是免费的，效果大概不会很好，以后肯定是要换的。

　　后期打算更换博客的主题为 NexT 或者 Butterfly，在这两个主题的基础上进行一些自定义的更改，实现更多的功能。不过，这都是很以后的事情了。

　　在内容方面呢，前期主要写一些以前学过的知识，比如刷机教程，安卓 root，电脑系统安装之类的。后期会写一些考研中学到的一些东西，主要是笔记。

随便聊聊

　　写博客，最重要的原因，大概是我无聊了，孤单了，寂寞了。我开始想念在学校的日子，那里有很多朋友。就像我一个室友说的那样，在寝室里，就算大家各做各的，也不会感到孤独。在校时有多么厌恶那个学校，现在就有多么想念。

　　我素来是不喜欢用微信聊天的，我看不到对面的人，不知道他说这句话是什么语气，什么表情，不清楚他是否愿意继续聊下去，我又是否有打扰到人家。还记得我上一次单独跟人在微信上闲聊还是 11 号。我自然也不敢先开口：有没有空？陪我聊会天？

　　电子通讯的普及，究竟是对人们情感之间联系的加强还是削弱？有很多人，渐渐地失去了联系，有很多事，慢慢地淡出了记忆。或许没有忘记，只是不再提起。

　　说到底，不敢主动跟别人聊天大概只是我的个人问题，需要克服。祝我早日脱离自闭生活，走向精彩人生。
